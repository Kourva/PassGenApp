# PassGenApp

Easy To Use GUI Password Generator App
> Made with **KivyMD**

# Install
+ clone
```bash
git clone https://github.com/MemeSec/PassGenApp && cd PassGenApp & chmod +x install.sh
```
+ requirements
```bash
./install.sh
```
+ run
```bash
python3 app.py
```
